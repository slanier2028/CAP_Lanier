Growth


My first attempt at trying to code was when I had to create a cooking app. I had to rely on using the OnEvent property for all of my code in this app due to me not knowing how to use any other properties. This caused me to experience some difficulties when trying to make some of the things in my cooking app function properly because it needed other properties for it to be able to work.


As the year went by, I began working on other projects like quizzes, images, games, and websites as well as actively participating in coding exercises which expanded my knowledge on how to use certain blocks of code such as using variables to store data, if / else statements, functions, etc. As my knowledge on coding grew, I managed to create more complex projects that could fully function as intended with very little difficulties.


Using everything that I have learned throughout Intro to Computer Science, I would go back to my cooking app to make some improvements like making some of the interactions function so that the app could be fully interactable.
